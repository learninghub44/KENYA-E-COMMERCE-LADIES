# Operations Feature

Platform operations tooling for production management.

## Capabilities

- **Maintenance Mode** — global, read-only, and scheduled maintenance with admin bypass
- **Rate Limiting** — per-user, per-seller, per-admin, per-API, per-IP limits
- **Health Engine** — database, storage, cache, queue, and external service health checks
- **Readiness Checks** — environment variables, connectivity, startup validation
- **System Diagnostics** — comprehensive health reports for all platform services
