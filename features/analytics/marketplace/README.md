# Marketplace Analytics

Agent 11B owns administrator-facing marketplace analytics. This feature consumes aggregate
marketplace data for super admin and admin users only.

Owned paths:

- `lib/analytics/marketplace`
- `components/dashboard/admin`
- `app/admin/analytics`
- `docs/analytics/marketplace`

This module does not implement search, review, messaging, notification, export, forecasting,
event analytics, or operational BI insights. Those belong to later analytics phases.
