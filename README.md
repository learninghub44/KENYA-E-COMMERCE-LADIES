wellness — built for scale (1M+ users, 100K+ sellers, 20M+ products).

## Stack

Next.js · React · TypeScript · Tailwind CSS · shadcn/ui · TanStack Query · Supabase
(Postgres + RLS + Edge Functions) · Cloudinary · Didit KYC · Google Analytics · Cloudflare

## Quick Start

```bash
pnpm install
cp .env.example .env.local
pnpm dev
```
