# Changelog

All notable changes to Zuri Market are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project follows [Semantic Versioning](https://semver.org/)
(`MAJOR.MINOR.PATCH`).

## [1.0.0] — TBA

### Added

- Complete brand identity with logos, color palette, and guidelines
- Proprietary legal documentation: LICENSE, COPYRIGHT.md, TRADEMARK.md,
  TERMS_OF_USE.md, OWNER.md, AUTHORS.md
- GitHub community and governance documentation: CONTRIBUTING.md,
  SECURITY.md, SUPPORT.md, GOVERNANCE.md, CODE_OF_CONDUCT.md, ROADMAP.md
- GitHub repository configuration: issue templates, pull request template,
  discussion template, CODEOWNERS, FUNDING.yml
- Production documentation: 12 guides covering deployment, DR, monitoring,
  scaling, security, incident response, troubleshooting
- Documentation project guides: overview, architecture, development,
  deployment, maintenance, scaling, security, repository standards

## [0.8.0] — Notifications & Platform Communications

### Added

- Durable event bus for platform-wide notification events
- User notification preferences
- Transactional email delivery via Resend
- Admin broadcast messaging with role-based access control
- Full test coverage for the notification pipeline
